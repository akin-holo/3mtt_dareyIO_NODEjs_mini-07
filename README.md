# Real-Time Chat App

A simple scalable chat application built with Node.js, Express, and Socket.IO.

## Features
- Real-time messaging
- Unique usernames
- Supports multiple concurrent users

## Getting Started

1. unzip the folder  
2. Install dependencies:  
   `npm install`  
3. Start the server:  
   `node server/index.js`  
4. Open browser at:  
   `http://localhost:3000`

## Tech Stack
- Node.js
- Express
- Socket.IO
